# ARSW BLUEPRINTS HeavyClient
## Compile and run instructions

* Para cada una de las pruebas se implementó **JUnit**, **Maven** y **spring-boot**.
* Proyecto creado en **IntelliJ IDEA**.

**Compiling**
```
$ mvn clean compile
```
**Packing**
```
$ mvn package
```    
**Execute**
```
$ mvn spring-boot:run"
```
